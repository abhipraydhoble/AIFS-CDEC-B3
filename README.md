# Application: 
- Application is a software program designed to solve user's problem.

# types of applications:
  -> Desktop Application
     - installed on computer
     - ex. ms word,  paint, vlc player

  -> Mobile Application
    - installed on Mobile phones
    - ex. instagram, whastapp, gpay

  -> Web Application
  - runs inside browser
  - ex. gmail, netflix, aws console


# monolithic 
- entire app is deployed as a single unit
# microservises
-  app deployend into small independent servises


# SDLC: software development life cycle
1. Client Requirement
  - Client needs a clothing store website/app
  - user can virtually try new cloths

2. Planing
  - timeline 
  - budget
  - tech stack 
  - features

3. Design
-  ui/ux team design screeens
-  architecture diagram 

4. Code
  # frontend:
    tech: html,css, js, react 
  # backend: 
     tech: springboot
  # database: 
      tech: mongodb, sql

5. Testing(QA)
- find bugs
- check performance
- security

6. Deployement(Live)
   - devops team:
     tech: OS,aws(cloud),docker,k8s,jenkins

7. Maintainance & Suppport
 - monitor app
 - fix bugs
